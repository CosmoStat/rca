proxs¶

Defines proximal operators to be fed to ModOpt algorithm that are specific to RCA (or rather, not currently in modopt.opt.proximity).

class proxs.KThreshold(iter_func)¶

This class defines linewise hard-thresholding operator with variable thresholds.

iter_func: function
Input function that calcultates the number of non-zero values to keep in each line at each iteration.
cost(x)¶

Returns 0 (Indicator of \(\Omega\) is either 0 or infinity).

op(data, extra_factor=1.0)¶

Return input data after thresholding.

reset_iter()¶

Set iteration counter to zero.

class proxs.LinRecombine(A, filters, compute_norm=False)¶

Multiply eigenvectors and (factorized) weights.

class proxs.StarletThreshold(threshold, thresh_type='soft')¶

Apply soft thresholding in Starlet domain.

threshold: np.ndarray
Threshold levels.
thresh_type: str
Whether soft- or hard-thresholding should be used. Default is 'soft'.
op(transf_data, **kwargs)¶

Applies Starlet transform and perform thresholding.
